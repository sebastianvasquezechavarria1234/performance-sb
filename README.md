# Performance SB 🏋️‍♂️

![Performance SB](https://img.shields.io/badge/Status-Development-orange)
![License](https://img.shields.io/badge/License-Proprietary-red)

**Performance SB** es una plataforma integral de gestión para centros de rendimiento deportivo y gimnasios. Diseñada para optimizar la interacción entre el centro, sus instructores y los estudiantes, proporcionando una experiencia fluida y profesional en el seguimiento del progreso físico y la gestión de servicios.

## 👥 Perfiles del Sistema

La plataforma está estructurada para ofrecer experiencias personalizadas según el tipo de usuario:

### 🎓 Estudiante
* **Gestión de Clases:** Visualización de horarios y clases inscritas.
* **Historial de Compras:** Seguimiento detallado de membresías y productos adquiridos.
* **Panel Personal:** Acceso a configuraciones de perfil y progreso personal.

### 🧘‍♂️ Instructor
* **Control de Alumnos:** Gestión y visualización de la lista de estudiantes asignados.
* **Programación:** Administración de sus propias clases y horarios.
* **Seguimiento de Ventas:** Visualización de detalles de compras relacionadas con sus servicios.
* **Perfil Profesional:** Gestión de información personal y profesional.

### 👤 Cliente
* **Landing Page Interactiva:** Exploración de servicios, planes y productos.
* **Proceso de Compra:** Experiencia optimizada para la adquisición de membresías y servicios del centro.
* **Gestión de Cuenta:** Registro y administración de datos básicos de usuario.

## 🛠️ Tecnologías Utilizadas

Este proyecto utiliza un stack moderno para garantizar alto rendimiento y una interfaz de usuario premium:

* **Frontend:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
* **Estilos:** [Tailwind CSS 4](https://tailwindcss.com/) (Diseño responsivo y moderno)
* **Animaciones:** [Framer Motion](https://www.framer.com/motion/) & [Canvas Confetti](https://www.kirilv.com/canvas-confetti/)
* **Iconografía:** [Lucide React](https://lucide.dev/)
* **Visualización de Datos:** [Recharts](https://recharts.org/)
* **Comunicación:** [Axios](https://axios-http.com/)
* **Componentes UI:** Radix UI, Swiper (Carousels), Sonner & SweetAlert2 (Notificaciones)

## 🚀 Instalación y Uso

### Requisitos previos
* Node.js (versión 18 o superior)
* npm o yarn

### Pasos para ejecutar localmente
1. Clonar el repositorio.
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```

## 🐳 Despliegue

El proyecto cuenta con soporte para **Docker**, incluyendo configuraciones de Nginx para un despliegue eficiente en entornos de producción.
